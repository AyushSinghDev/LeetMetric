# LeetMetric 

A modern, interactive web dashboard to visualize your LeetCode problem-solving statistics — total problems solved, difficulty distribution, progress over time, and more.

Live Demo : [https://leet-metric-devxt.vercel.app/](https://leet-metric-devxt.vercel.app/)  

---

## What is LeetMetric

LeetMetric helps you track and understand your LeetCode journey by providing a clean UI with accurate metrics.
Enter your LeetCode username in the input field to fetch and view your stats in real time.

- Total problems solved  
- Distribution by difficulty (Easy / Medium / Hard)  
- Progress over time (e.g. solved per day/week/month)  
- Contest Rating
